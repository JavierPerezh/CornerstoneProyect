# CornerstoneProyect

Sistema modular de asistencia inteligente para madres en etapa posparto, implementado como un dispositivo físico con interfaz conversacional y una infraestructura backend segura. El sistema captura voz, realiza transcripción automática, extrae información clínica estructurada mediante modelos de lenguaje, y evalúa niveles de riesgo utilizando un enfoque híbrido basado en reglas deterministas verificables y un modelo estadístico interpretable (regresión logística).

La arquitectura prioriza transparencia, trazabilidad y privacidad de los datos mediante una base de datos relacional y separación estricta entre componentes conversacionales y componentes de decisión clínica.

El repositorio contiene los módulos de hardware, procesamiento de lenguaje natural, motor de evaluación de riesgo, backend API, base de datos y frontend de visualización.
